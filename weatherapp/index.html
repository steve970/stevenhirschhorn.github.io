PROJECT NAME: Weather App for Dogs

weather app + puppies, because they obviously go together.

Using a basic Express server for routing and handling the OpenWeatherMap API key, I created a weather website, aimed at dog lovers, specifically puggles. Type in the city and state of the where you'd like to know the weather and get a fun weather report and temperature.

JavaScript, jQuery, Express, FlickerAPI, OpenWeatherMap, freegeoip
